- https://placeholder.com

## Image Optimization Tips

1. **Compress images** before uploading (use TinyPNG or ImageOptim)
2. **Use descriptive filenames** (lowercase, hyphens)
3. **Keep file sizes under 500KB** for web performance
4. **Save in WebP format** for best performance (or JPG as fallback)
